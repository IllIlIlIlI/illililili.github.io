CONFIG SERVICES

SMTP

Configurations for Simple Mail Transfer Protocol service

Description

The SMTP service will simulate a SMTP server.

Syntax

[service.<nickname_of_smtp>]
type="smtp"

Arguments in config file

There is 1 argument in smtp service:

argument default setting explanation
banner SMTPd Set the banner of your SMTP server.

Example

[service.smtp]
type="smtp"

[[port]]
port="tcp/25"
services=["smtp"]

How to test

telnet 127.0.0.1 25

Availability


Linux MacOS
smtp yes yes
